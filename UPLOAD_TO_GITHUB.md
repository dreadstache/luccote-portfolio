# Upload This Website to GitHub

Repository:

`https://github.com/Dreadstache/luccote-portfolio`

## What to upload

Upload the CONTENTS of this folder:

- `index.html`
- `assets`
- `projects`
- `README.md`
- `.nojekyll`

Do not upload the outer `Luc_Cote_CareerOS_Web_v1` folder as one nested folder.

## Browser-only upload

1. Open the repository on GitHub.
2. Click **Add file**.
3. Click **Upload files**.
4. Drag everything listed above into the upload area.
5. Commit message: `Publish CareerOS Web v1`
6. Commit directly to `main`.
7. Open **Settings → Pages**.
8. Choose **Deploy from a branch**.
9. Choose `main` and `/ (root)`.
10. Save.

Your temporary site address will be:

`https://dreadstache.github.io/luccote-portfolio/`

## Before connecting luccote.com

Confirm the temporary GitHub Pages address works. Keep Google Sites connected until the GitHub version is verified.

## Editing later

For a simple text change:

1. Open the file on GitHub.
2. Click the pencil icon.
3. Edit the text.
4. Commit the change.

For larger changes, replace the files with an updated CareerOS Web package.
